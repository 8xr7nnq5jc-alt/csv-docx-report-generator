# Portfolio and outreach copy

## Portfolio - English

CSV to DOCX Report Generator is a small internal web app that lets a user upload CSV data and automatically generate formatted Word reports. It includes a simple login, report type selection, a direct download flow and activity logging. The project demonstrates how existing scripts can be made accessible through a clear browser workflow.

## Portfolio - Nederlands

CSV to DOCX Report Generator is een kleine interne webapp waarmee een gebruiker CSV-data kan uploaden en automatisch nette Word-rapporten kan genereren. De app bevat een eenvoudige login, keuze tussen verschillende rapporttypes, direct downloaden en een activity log. Het project laat zien hoe bestaande scripts toegankelijk kunnen worden gemaakt via een duidelijke webinterface.

## Klantbericht

Hoi, ik heb alvast een kleine werkende demo gemaakt van de flow: een simpele login, CSV-upload, keuze tussen een individual- of teamrapport, automatische Word-download, activity log en een nette webinterface. Ik kan dit verder aanpassen op jullie bestaande scripts, templates en deploymentvoorkeur.

## Reddit/LinkedIn post

Ik heb een kleine Node.js-webapp gebouwd die CSV-uploads omzet naar Word-rapporten. Handig voor teams die nog met losse scripts werken maar een eenvoudige webinterface nodig hebben. De demo bevat login, keuze tussen individual/team reports, DOCX-download en een activity log.
