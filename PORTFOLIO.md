# Portfolio and outreach copy

## Portfolio - English

CSV to DOCX Report Generator is a small internal web app that lets a user upload CSV data and automatically generate formatted Word reports. It includes a simple login, report type selection, a direct download flow and activity logging. The project demonstrates how existing scripts can be made accessible through a clear browser workflow.
